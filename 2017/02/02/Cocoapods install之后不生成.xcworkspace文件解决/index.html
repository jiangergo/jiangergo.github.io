Cocoapods install之后不生成.xcworkspace文件解决

2017-02-02

首先，祝大家新年快乐，鸡年大吉！


感觉有一段时间没有动电脑了，良心不安，于是决定写个 Demo 练练手。
结果发现 cocoapods 执行了pod install 之后没有生成 .xcworkspace 文件。

这样：

错误提示：

localhost:Test-Crash Jiangergo$ pod init
localhost:Test-Crash Jiangergo$ pod install --no-repo-update
Analyzing dependencies
Downloading dependencies
Installing AFNetworking (3.1.0)
Generating Pods project
Abort trap: 6


解决方案：

Removing all cocoapods gems：

sudo gem uninstall cocoapods
sudo gem uninstall cocoapods-core
sudo gem uninstall cocoapods-deintegrate
sudo gem uninstall cocoapods-downloader
sudo gem uninstall cocoapods-plugins
sudo gem uninstall cocoapods-search
sudo gem uninstall cocoapods-stats
sudo gem uninstall cocoapods-try
sudo gem uninstall cocoapods-trunk

Then installing latest pre release version:

sudo gem install cocoapods --pre

显示以下，成功！

Done installing documentation for nanaimo, claide, 
CFPropertyList, activesupport, xcodeproj, ruby-macho, 
molinillo, gh_inspector, fourflusher, cocoapods-trunk, 
cocoapods-downloader, cocoapods-deintegrate, cocoapods-
core, cocoapods after 16 seconds
14 gems installed

赶快试试 pod install 吧！
